방구석 실험실은 과학 실험을 체험하기 어려운 학교 밖 청소년들을 위한 VR 과학 실험실입니다. <br>
학교를 다니지 않아, 배움의 기회를 놓쳐 과학 실험을 실제로 접하지 못하는 `학교 밖 청소년`들이 있습니다. 이들에게 가상의 VR 실험실 환경을 제공함으로써 시간, 장소, 예산 등의 요소에 영향받지 않고 자유롭게 과학 실험을 체험하고 눈으로 확인할 수 있습니다.

## 🗓️ 개발 기간

- `2022.10.10 ~ 2022.11.21 (6주)` : 프로젝트 기획, 설계, 개발, 발표
- 2022.11.18 : 미추홀구 꿈드림 센터에서 5명의 학교 밖 청소년들에게 [방구석 실험실] 시연
- 2022.12.03 ~ 2022.12.06 (리팩토링) : 미추홀구 꿈드림에 기부 목적으로 *서버리스 버전 개발
* 서버리스(serverless)란 개발자가 서버를 관리할 필요 없이 애플리케이션을 빌드하고 실행할 수 있도록 하는 클라우드 네이티브 개발 모델입니다.

<br>

## 💡 주요 기능

- 중학교 과학 교과서에서 선정한 실험을 Unity를 활용하여 실제 실험과 비슷하고 생동감 있게 구현 
- VR 기기 `오큘러스트 퀘스트2`에 APK를 주입하여 실행
- `실험 맵`에서는 중학교과과정 중 총 9개의 실험을 선정하여 체험할 수 있도록 제작하였습니다.
- `몸속 맵`에서는 입에서부터, 식도, 위까지 버스를 타고 이동하여 음식의 소화과정을 관찰 및 각 소화 기관의 역할을 체험할 수 있습니다.
- `우주 맵`에서는 태양계 행성들의 자전과 공전, 각 행성들의 특성을 관찰할 수 있습니다.
